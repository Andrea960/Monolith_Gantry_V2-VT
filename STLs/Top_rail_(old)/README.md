Top rail requires mirrored parts in some cases.

**AB** = normal, **CD** = mirrored

- Toolhead with the right front belt on top: **AB**
- Toolhead with the left front belt on top: **CD**

|AB|4WD|AB|AB|AB|AB|
|CD|2WD|AB|AB|AB|CD|
